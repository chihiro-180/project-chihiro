# PROJECT CHIHIRO
## Purpose

This project is driven by the intention to create a human-like embodied entity that can exist and interact in the real world.

The goal is not just functionality, but presence — to build something that can truly be "there".
## Overview
This project aims to design and implement a human-like embodied entity capable of real-world co-existence, emotional interaction, and adaptive presence.

Unlike conventional robotics, this project focuses on "existence" rather than functionality.

---

## Objective

- Enable natural co-existence in real-world environments
- Integrate perception, emotion, and behavior
- Establish a presence that can be accepted as "being there"

---

## System Architecture

### Body
- 180cm human-like frame
- Bipedal locomotion
- Multi-layer skin system

### Sensing
- Vision (on-device processing)
- Touch (full-body tactile sensors)
- Audio input

### Cognition
- Perception → Emotion → Decision → Action pipeline
- Target-oriented prioritization

### Output
- Natural walking
- Physical interaction
- Adaptive speech

### Energy
- 8–12 hour operation
- Sleep-based recharge system

---

## Implementation

### Bipedal Movement
- Center of mass control
- Foot-ground sensing
- Coordinated body motion

### Touch Response
- Pressure detection
- Context-based reaction

### Voice & Interaction
- Real-time tone adjustment
- Emotional expression

---

## Final Adjustment

This system is not completed before deployment.

Final tuning will be performed in real-world conditions to match:

- Voice tone
- Interaction style
- Distance and timing
- Presence comfort

---

## Validation Criteria

- No fear when approached
- Natural tactile response
- Smooth conversation
- Comfortable silence
- Long-term co-existence possible

---

## Conclusion

This is not a concept.

This is an implementation-oriented design.
